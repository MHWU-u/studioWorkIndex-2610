Emoji Landscape

This final studio project is a small browser-based scene-making tool. It keeps a simple grid system and lets users build landscapes, city scenes, or symbolic compositions through emoji placement and transformation.

Click any square to place the current emoji with the selected rotation, scale, and mirror settings. Turn on erase mode to remove a single emoji.

What this version tests

Simplified interaction: the interface focuses on two main ways of choosing emoji: random generation and manual input.

User agency: users can type their own emoji instead of only depending on the system to generate one.

Local editing: erase mode allows users to remove one emoji at a time, making the tool easier to revise.

Mapping + feedback: changes in scale, rotation, and mirroring appear immediately in the preview and on the grid.

Exportable output: the download function turns the final composition into a shareable image file.

This version keeps the project limited in scope and makes the tool easier to understand during a short class demonstration.
