Final Studio Project

Emoji Landscape

Emoji Landscape is a small browser-based creative tool for building symbolic landscapes, city scenes and visual compositions with emoji. Instead of drawing with lines or brushes, users choose or randomise emoji, transform them, place them on a 9×9 grid, revise individual cells and export the final image.

The project treats emoji as a shared digital visual language. It explores how simple symbols can become building blocks for image-making. The main interaction principles are mapping and feedback: rotation, scale, mirror and theme controls immediately change what the user sees in the preview and on the grid.

Project Development

Conceptual response

This project responds to the studio brief by creating a small interactive expressive tool for web browsers. It uses emoji as a symbolic material for image-making, rather than as decoration in messages. The tool is designed to be playful, accessible and open-ended.

Technical approach

The final tool is built with HTML, CSS and JavaScript. The interface uses a clear control panel, a live preview, a structured grid and simple export functionality. Code comments in the final project explain the design choices, including conceptual response, technical approach, characterisation, learnability and usability.

Design decisions

The 9×9 grid is kept as a deliberate limitation. It gives users a clear structure and reduces the difficulty of deciding exact freehand placement. The final version also removes the earlier category buttons and emoji palette so the interaction is simpler and easier to learn.

Peer feedback response

Peer user testing showed that the tool worked well as a simple creative system, but users wanted more control and easier editing. In response, the final version adds manual emoji input, single-cell erase mode and a download function. These changes make the tool more usable while keeping the project limited in scope.
