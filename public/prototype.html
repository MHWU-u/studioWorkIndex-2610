What this version tests

Expanded visual vocabulary: more natural and city emojis support scene-building rather than only isolated symbols.

User agency: users can now type or select their own emoji instead of only using random generation.

Local editing: erase mode allows users to remove one emoji at a time, making the tool easier to revise.

Mapping + feedback: changes in scale, rotation, and mirroring appear immediately in the preview and on the grid.

Exportable output: the download function turns the final composition into a shareable image file.

This version keeps the project limited in scope, but makes the tool more useful for building small landscapes, mixed environments, and symbolic city scenes.
